# Lab Task 7: Smart Home System (Inheritance + Aggregation + Static Members)

## Objective

The goal of this lab is to implement:

Single inheritance

Multilevel inheritance

Hierarchical inheritance

Aggregation (has-a relationship)

Static class members

Unique ID generation

Proper encapsulation

Object state management

You will build a Smart Home system that stores different types of appliances. Each appliance must have a unique automatically generated ID.

IMPORTANT: You are NOT required to implement public static void main(). Testing will be done separately.
